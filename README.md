# 🛍️ ShopSense AI

**ShopSense AI** is a centralized AI-powered e-commerce platform built as our Final Year Project. It provides a unified marketplace experience by integrating multiple online stores into a single platform, allowing users to compare product prices, availability, and ratings across various sellers — all in one place.

Beyond basic comparison, ShopSense AI leverages **Artificial Intelligence** to enhance user experience with features like **NLP-based smart search** and **personalized product recommendations**, making online shopping faster, smarter, and more intuitive.

## ✨ Features

- 🔄 **Centralized E-Commerce Platform**  
  Aggregate and compare products, prices, and availability from multiple online stores in real-time.

- 📝 **NLP-Based Product Search**  
  Use natural language queries like _"show me affordable gaming laptops"_ and get relevant, intelligent results.

- 🤖 **AI-Powered Recommendations**  
  Smart product recommendations based on browsing history, preferences, and popular trends.

- 📊 **Price Comparison Engine**  
  Compare prices and discounts from various stores on a single product page.

- 💬 **Product Review Summarization** (Planned)  
  AI-generated summaries of customer reviews for quick insights.

## 🖥️ Tech Stack

- **Frontend:** Angular, HTML5, CSS3, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI/NLP:** Python (NLTK / spaCy / Transformers)
- **APIs:** Custom Store APIs, Web Scraping Utilities
